# Lab17-18

# Назначение программы
Приложение для управления зоопарками и животными в них. Программа позволяет вести учет зоопарков, животных через графический интерфейс.

# Реализованные функции
- Добавление / удаление зоопарков
- Добавление / удаление животных из самих зоопарков и из общего списка животных
- Обновление зоопарка
- Обновление животного

# Как запустить проект
1. Скачать файл с репозитория
2. Открыть файл с названием "Lab17-18.sln" в Visual Studio
3. Настроить подключение к базе данных в файле конфигурации
4. Запустить проект (F5)
